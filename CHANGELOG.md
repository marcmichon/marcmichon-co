# Changelog

Toutes les évolutions importantes de MM Security Intelligence sont documentées dans ce fichier.

---

## [1.3.0] - 2026-07-22

### 🚀 Added

- Introduced a lightweight internationalization (i18n) layer with French and English translation dictionaries.
- Added reusable translation helpers across the application.
- Prepared the project architecture for multilingual support.

### ✨ New

- New **MM Security Intelligence** presentation.
- New **Vulnerability Explorer** product experience.
- New executive intelligence report generation.
- New business impact assessment.
- New confidence assessment.
- New explainable decision engine.
- New operational response plan.
- New normalized PDF export.

### 🌍 Localization

- Fully translated the Vulnerability Explorer interface into French.
- Localized executive summaries, operational intelligence and response plans.
- Localized PDF report.
- French date formatting.
- French percentage formatting.
- French severity labels.
- French confidence labels.
- French export workflow.
- French business impact messages.
- French analyst recommendations.

### 🔧 Changed

- Renamed the public product from **MM Security Intelligence Labs** to **Boîte à outils de cybersécurité**.
- Renamed **Vulnerability Explorer** to **Explorateur de vulnérabilités** for French users.
- Improved overall wording across the product.
- Harmonized terminology between UI and PDF export.

### 🐛 Fixed

- Fixed PDF export always using the first searched CVE.
- Fixed PDF export event binding after multiple searches.
- Fixed several inconsistent English labels.
- Improved confidence rendering.
- Improved report normalization.
